# B2B Buyer Lead Extractor

Extract SaaS buyer intelligence and firmographic data for account-based marketing. Identify companies actively buying software and enrich with decision-maker contacts.

## Features

- Identify companies with active SaaS buying signals
- Extract firmographic data (size, revenue, industry, tech stack)
- Find decision-maker contacts and titles
- Buying intent signals and technology usage data
- Filter by company size, industry, and location
- CRM-ready output format for sales teams
